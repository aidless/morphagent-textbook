# 实验 27：跨领域迁移实验

> 状态：可运行（Mock 模式）
> 目标：评估 Agent 从源领域到目标领域的知识迁移能力
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

模拟 Agent 的跨领域迁移：在源领域训练后，迁移到不同距离的目标领域。
测量迁移效率和领域距离对表现的影响。

## 实验步骤

1. 定义 3 个源领域和 5 个目标领域
2. 计算领域间距离（基于任务特征）
3. 模拟直接迁移 vs 微调迁移 vs 从头训练
4. 分析迁移效率与领域距离的关系

## 评测指标

- 迁移后任务完成率
- 迁移效率（vs 从头训练）
- 领域距离与性能损失的相关性

cd experiments/exp-27-cross-domain
