# 实验 26：L0->L5 能力等级跃迁

> 状态：可运行（Mock 模式）
> 目标：模拟 MorphAgent 从 L0 到 L5 的能力等级跃迁过程
> 难度：★★★★★
> 预计时间：8 小时

## 实验目的

定义 MorphAgent 的 6 个能力等级（L0~L5），模拟每个等级的核心能力表现。
验证等级跃迁的必要条件和阈值。

## 实验步骤

1. 定义 L0~L5 的能力维度和阈值
2. 为每个等级生成模拟表现数据
3. 识别等级跃迁的关键能力指标
4. 可视化等级跃迁路径

## 评测指标

- 各等级核心指标得分
- 跃迁必要条件满足率
- 能力差距分析

## 运行方式

```bash
cd experiments/exp-26-level-progression
python main.py
```
