# 实验 29：消融实验

> 状态：可运行（Mock 模式）
> 目标：通过消融实验量化各子系统的独立贡献
> 难度：★★★★★
> 预计时间：8 小时

## 实验目的

通过逐个移除子系统，量化每个子系统的独立贡献。
验证"完整系统 > 任何子集"的核心假设。

## 实验步骤

1. 定义完整系统（P+T+M+C 四子系统）
2. 设计 6 种消融配置（逐个移除 + 两两移除）
3. 在 5 个任务上测试每种配置
4. 计算每个子系统的增量贡献

## 评测指标

- 各消融配置的任务完成率
- 单个子系统的增量贡献
- 子系统重要性排序

## 运行方式

```bash
cd experiments/exp-29-ablation
python main.py
```
