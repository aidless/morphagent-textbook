# 实验 17：元控制器评估

> 状态：可运行（Mock 模式）
> 目标：评估元控制器对子系统调度的有效性
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

实现元控制器（Meta-Controller），统一调度 Prompt/Tool/Memory/Cognition 四个子系统。
对比有无元控制器的任务完成效率和资源消耗。

## 实验步骤

1. 定义 4 个子系统的能力模型
2. 实现元控制器的调度策略（固定权重 / 自适应 / 强化学习模拟）
3. 模拟 8 个复合任务，测量完成率和资源消耗
4. 对比有无元控制器的表现差异

## 评测指标

- 任务完成率
- 子系统调用次数（资源消耗）
- 调度均衡度（子系统利用率方差）

## 运行方式

```bash
cd experiments/exp-17-meta-controller
python main.py
```
