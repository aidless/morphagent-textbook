# 实验 30：可复现性检验

> 状态：可运行（Mock 模式）
> 目标：检验实验结果的可复现性，评估随机种子和参数敏感性
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

通过多次重复运行同一实验，评估结果的稳定性和可复现性。
分析随机种子和关键参数对结果的影响。

## 实验步骤

1. 选取 3 个核心实验进行复现检验
2. 使用不同随机种子运行 10 次
3. 计算结果的均值、标准差、变异系数
4. 分析关键参数的敏感性（温度、top_p、max_steps）
5. 输出可复现性报告

## 评测指标

- 变异系数（CV）
- 置信区间宽度
- 参数敏感性排序

## 运行方式

```bash
cd experiments/exp-30-reproducibility
python main.py
```
