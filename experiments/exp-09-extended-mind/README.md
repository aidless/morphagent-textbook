# 实验 9：延展心智实验

> 状态：可运行（Mock 模式）
> 目标：验证"外部工具可作为 Agent 认知的延伸"（Clark & Chalmers 的 Parity Principle）
> 难度：★★★☆☆
> 预计时间：3 小时

## 实验目的

模拟 Otto & Inga 思想实验的 AI 版本，验证外部工具（记事本、计算器、搜索引擎）
作为认知延伸的有效性。

## 实验步骤

1. 实现 3 个外部工具：笔记本、计算器、搜索引擎
2. 设计 5 个需要外部工具的任务
3. 对比有外部工具 vs 无外部工具的 Agent 表现
4. 用 Parity Principle 分析结果

## 预期结果

| 条件 | 平均成功率 |
|---|---|
| 无外部工具 | ~45% |
| 有笔记本 | ~65% |
| 有计算器 | ~70% |
| 有全部工具 | ~90% |
