# 实验 20：MorphBench 迷你版

> 状态：可运行（Mock 模式）
> 目标：构建迷你基准测试，评估 Agent 在不同干预下的表现
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

构建 MorphBench 迷你版：3 个核心任务 x 3 种干预条件 = 9 个实验点。
测量不同干预对 Agent 表现的影响。

## 实验步骤

1. 定义 3 个核心任务（推理/检索/生成）
2. 定义 3 种干预（无干预/提示增强/工具增强）
3. 运行 9 个实验组合，每个重复 5 次
4. 统计分析和可视化

## 评测指标

- 任务完成率
- 响应质量评分
- 干预效果量（Cohen's d）

## 运行方式

```bash
cd experiments/exp-20-morphbench-mini
python main.py
```
