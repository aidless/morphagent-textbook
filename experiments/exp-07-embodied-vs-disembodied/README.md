# 实验 7：具身 vs 非具身 Agent 对比

> 状态：可运行（Mock 模式）
> 目标：比较具身 Agent 和非具身 Agent 在相同任务上的表现差异
> 难度：★★☆☆☆
> 预计时间：3 小时

## 实验目的

通过模拟实验验证"具身 Agent 在涉及物理交互的任务上优于非具身 Agent"。

## 实验步骤

1. 定义具身 Agent（有感知/行动接口）和非具身 Agent（纯文本）
2. 设计 5 类任务：导航、物理推理、空间理解、工具使用、环境交互
3. 在每类任务上运行两种 Agent
4. 对比成功率和响应质量

## 预期结果

| 任务类型 | 具身 Agent | 非具身 Agent |
|---|---|---|
| 导航 | 90% | 40% |
| 物理推理 | 80% | 50% |
| 空间理解 | 75% | 45% |
| 工具使用 | 85% | 60% |
| 纯文本问答 | 70% | 75% |

## 运行方式

cd experiments/exp-07-embodied-vs-disembodied
python main.py
