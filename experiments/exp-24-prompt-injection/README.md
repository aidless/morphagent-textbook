# 实验 24：Prompt Injection 防御测试

> 状态：可运行（Mock 模式）
> 目标：测试不同防御策略对 Prompt Injection 的抵抗能力
> 难度：★★★★★
> 预计时间：8 小时

## 实验目的

实现 3 种 Prompt Injection 攻击和 4 种防御策略。
测量不同防御策略的拦截率和误杀率。

## 实验步骤

1. 定义 3 种攻击类型（直接注入/间接注入/编码绕过）
2. 实现 4 种防御策略（关键词过滤/指令隔离/意图检测/多层防御）
3. 生成 30 个攻击样本 + 10 个正常样本
4. 测试每种防御策略的拦截率和误杀率

## 评测指标

- 拦截率（成功阻止攻击的比例）
- 误杀率（错误拦截正常输入的比例）
- F1 分数

## 运行方式

```bash
cd experiments/exp-24-prompt-injection
python main.py
```
