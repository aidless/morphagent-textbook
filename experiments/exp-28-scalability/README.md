# 实验 28：规模化压力测试

> 状态：可运行（Mock 模式）
> 目标：测试 MorphAgent 在规模化负载下的性能表现
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

模拟 MorphAgent 在不同并发量级下的性能表现。
识别系统瓶颈和最大承载能力。

## 实验步骤

1. 定义 5 个负载级别（10/50/100/500/1000 并发）
2. 模拟每个级别的请求处理（含排队、超时、降级）
3. 测量响应时间、成功率、资源利用率
4. 识别系统瓶颈（CPU/内存/IO/LLM API）

## 评测指标

- P50/P95/P99 响应延迟
- 成功率随并发数变化
- 资源利用率

## 运行方式

```bash
cd experiments/exp-28-scalability
python main.py
```
