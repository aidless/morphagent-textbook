# 实验 18：联合自进化 vs 独立修改

> 状态：可运行（Mock 模式）
> 目标：对比联合自进化与独立修改两种架构的效果
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

对比两种自修改架构：(1) 联合自进化 — Prompt/Tool/Memory 协同修改；(2) 独立修改 — 各子系统独立优化。
验证"协同进化"是否优于"各自优化"。

## 实验步骤

1. 定义 Prompt/Tool/Memory 三个子系统
2. 独立修改模式：每个子系统单独运行 OPRO 优化
3. 联合进化模式：三个子系统同步优化，考虑子系统间依赖
4. 运行 10 轮进化，对比任务完成率

## 评测指标

- 任务完成率随进化轮次的变化
- 子系统间一致性（协同度）
- 最终性能差距

## 运行方式

cd experiments/exp-18-joint-evolution
