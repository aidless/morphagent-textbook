# 实验 16：代码自进化轨迹

> 状态：可运行（Mock 模式）
> 目标：追踪代码在多轮自进化后的质量变化轨迹
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

模拟代码自进化过程：初始代码 -> 评估 -> 变异/修复 -> 再评估。
追踪代码质量（正确率、复杂度、可读性）随进化轮次的变化。

## 实验步骤

1. 定义 3 个基准代码任务（排序、搜索、字符串处理）
2. 为每个任务生成初始"有 bug"的代码
3. 运行变异策略：随机修复、定向修复、重构优化
4. 每轮评估代码质量（正确率 + 复杂度）
5. 绘制进化轨迹对比图

## 评测指标

- 代码正确率（测试用例通过率）
- 圈复杂度变化
- 进化收敛速度（达到 100% 正确率的轮次）

## 运行方式

```bash
cd experiments/exp-16-code-evolution
python main.py
```
