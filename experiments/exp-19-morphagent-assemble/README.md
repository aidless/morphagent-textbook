# 实验 19：MorphAgent 5 子系统集成

> 状态：可运行（Mock 模式）
> 目标：模拟 MorphAgent 五个子系统的集成与协调运行
> 难度：★★★★★
> 预计时间：8 小时

## 实验目的

模拟 MorphAgent 的 5 个核心子系统（感知/决策/执行/记忆/学习）的集成。
测量子系统间通信开销和协同效率。

## 实验步骤

1. 定义 5 个子系统的接口和行为模型
2. 实现子系统间的消息传递机制
3. 运行 6 个集成任务，测量端到端延迟和成功率
4. 对比不同集成拓扑（星型/环型/全连接）

## 评测指标

- 端到端任务完成率
- 子系统间通信次数
- 平均响应延迟

## 运行方式

```bash
cd experiments/exp-19-morphagent-assemble
python main.py
```
