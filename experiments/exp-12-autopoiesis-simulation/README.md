# 实验 12：自创生模拟

> 状态：可运行（Mock 模式）
> 目标：模拟 Maturana & Varela 的 Autopoiesis（自创生）概念
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

实现一个简化的自创生系统模拟，验证"系统能够自我维持、自我修复、自我生产"的核心特征。

## 实验步骤

1. 实现一个由"组件"和"过程"组成的简单 autopoietic 系统
2. 定义 3 个核心过程：生产（组件合成）、修复（组件替换）、边界（系统维护）
3. 在不同环境扰动下观察系统的自创生行为
4. 测量系统的存活率和自修复能力

## 核心概念

- **自生产(Autopoiesis)**: 系统自己生产维持自身所需的组件
- **自修复**: 系统能修复被破坏的组件
- **自维持**: 系统在没有外部干预的情况下持续存在
