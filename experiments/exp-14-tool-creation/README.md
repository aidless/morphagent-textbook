# 实验 14：自动工具创建

> 状态：可运行（Mock 模式）
> 目标：模拟 Agent 根据任务需求自动创建新工具的过程
> 难度：★★★★☆
> 预计时间：5 小时

## 实验目的

实现 Agent 根据任务反馈自动识别缺失工具、创建新工具并验证有效性的过程。

## 实验步骤

1. 给 Agent 一组初始工具
2. 给定一系列任务，检测哪些任务因缺少工具而失败
3. 模拟 Agent 自动创建新工具（生成 name + description + mock 实现）
4. 用新工具重新运行失败任务，验证成功率提升

## 核心流程

失败检测 -> 工具需求分析 -> 工具创建 -> 验证 -> 迭代

## 运行方式

```bash
cd experiments/exp-14-tool-creation
python main.py
```
