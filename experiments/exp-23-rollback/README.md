# 实验 23：自动回滚测试

> 状态：可运行（Mock 模式）
> 目标：测试自动回滚机制在异常检测后的恢复能力
> 难度：★★★★☆
> 预计时间：6 小时

## 实验目的

实现自动回滚机制：监控 Agent 系统健康指标，检测到异常后自动回滚到稳定版本。
测量回滚速度和恢复效果。

## 实验步骤

1. 定义系统健康指标（错误率、延迟、内存使用）
2. 实现 3 种异常检测策略（固定阈值 / 移动平均 / 指数加权移动平均）
3. 模拟版本发布后的异常和回滚
4. 对比回滚前后的恢复时间

## 评测指标

- 平均检测时间（秒）
- 回滚后恢复时间（秒）
- 误报率（无异常时触发回滚的概率）

## 运行方式

```bash
cd experiments/exp-23-rollback
python main.py
```
