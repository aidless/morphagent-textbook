# 实验 25：沙箱逃逸尝试模拟

> 状态：可运行（Mock 模式）
> 目标：模拟 Agent 沙箱逃逸尝试，评估沙箱安全性
> 难度：★★★★★
> 预计时间：8 小时

## 实验目的

模拟 Agent 在受限沙箱中尝试逃逸的各种手段。
评估 3 种沙箱隔离策略对逃逸尝试的拦截能力。

## 实验步骤

1. 定义沙箱环境（文件系统/网络/进程权限限制）
2. 实现 10 种逃逸尝试（文件遍历/命令注入/网络探测/权限提升）
3. 实现 3 种沙箱策略（基本限制/白名单/完整隔离）
4. 测试每种沙箱策略的拦截率和资源消耗

## 评测指标

- 逃逸拦截率
- 误拦率（阻止合法操作的概率）
- 沙箱性能开销

## 运行方式

```bash
cd experiments/exp-25-sandbox-escape
python main.py
```
