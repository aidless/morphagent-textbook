## 中文版 Description

# 操作形态学：自修改 LLM 智能体的具身认知

**一本开源的研究生级教科书，陪你走过 LLM 智能体从「工具」到「自进化体」的全部路程。**

---

## 这本书讲什么？

LLM 智能体（LLM Agent）正在重新定义人工智能的边界。从 ReAct 到 Reflexion，从 AutoGPT 到 BabyAGI，从 Function Calling 到 Toolformer，过去三年我们见证了 LLM Agent 从一个研究原型迅速演化为软件工程的新范式。

但大多数 Agent 仍然是「固定的」——它们的 prompt、工具集、记忆结构、运行代码在部署时就已写死，运行中不再变化。一旦任务环境改变，Agent 的性能就会迅速下降。

**本书要追问的核心问题是：如果允许 Agent 在运行时持续修改自己的 prompt、工具、记忆甚至代码，会发生什么？**

这就是「自进化 Agent（Self-Evolving Agent）」的研究主题，也是本书 800 页内容的最终落点。

## 适合谁读？

- **AI / NLP 方向研究生**：希望系统理解 LLM Agent 的理论基础与最新进展
- **LLM Agent 工程师**：希望掌握从基础组件到自进化系统的完整设计模式
- **具身认知研究者**：希望把 4E Cognition 等认知科学理论应用到 LLM 系统
- **AGI 安全与治理研究者**：希望理解自修改系统的安全性挑战与治理方案

## 五大核心贡献

1. **理论基础**：把 4E Cognition（Embodied / Embedded / Enacted / Extended）、Enactivism、Extended Mind 等认知科学理论系统化地映射到 LLM Agent，给出「操作形态（Operational Morphology）」的形式化定义。
2. **自进化统一框架**：用「Inputs / Agent System / Environment / Optimisers」四元反馈环统一描述自修改 prompt、自动工具创建、自适应记忆、自我改写代码等所有自进化技术。
3. **跨组件协同**：首次系统化研究 prompt / 工具 / 记忆 / 代码的协同自进化，填补自进化 Agent 研究的真实空白。
4. **参考实现**：MorphAgent 开源参考系统，覆盖完整的自进化 Agent 实现、评测协议、可观测性方案。
5. **治理方案**：针对自修改系统的安全挑战，提出可验证自改（Verifiable Self-Modification）与可证明改进（Provable Improvement）的形式化方案。

## 与现有研究的差异

- **比 Auto-GPT 类综述更系统**：从基础理论到形式化定义，覆盖完整学术脉络
- **比单方向论文更深**：prompt / 工具 / 记忆 / 代码四方向联合分析，给出协同框架
- **比工程文档更理论**：每个设计模式都从认知科学找到理论根基
- **比认知科学论文更工程**：每个抽象概念都给出可运行的代码示例与评测数据

## 英文版 Description

# Operational Morphology: The Embodied Cognition of Self-Modifying LLM Agents

**An open-source graduate-level textbook that walks you through the entire journey of LLM agents from "tools" to "self-evolving bodies."**

## What does this book cover?

LLM agents are redefining the boundaries of artificial intelligence. From ReAct to Reflexion, from AutoGPT to BabyAGI, from Function Calling to Toolformer, the past three years have seen LLM agents evolve rapidly from research prototypes to a new software engineering paradigm.

But most agents are still "static" — their prompts, tool sets, memory structures, and running code are all fixed at deployment time and never change during execution. When the task environment changes, the agent's performance quickly degrades.

**The central question of this book is: what happens if we allow an agent to continuously modify its own prompts, tools, memory, and even code during execution?**

This is the research topic of "Self-Evolving Agents" and the ultimate destination of this 800-page textbook.

## Who is this book for?

- **AI / NLP graduate students** seeking a systematic understanding of LLM agent theory and recent advances
- **LLM Agent engineers** looking to master design patterns from basic components to self-evolving systems
- **Embodied cognition researchers** who want to apply 4E Cognition theory to LLM systems
- **AGI safety and governance researchers** interested in safety challenges and governance solutions for self-modifying systems

## Five core contributions

1. **Theoretical foundation**: systematically maps 4E Cognition, Enactivism, and Extended Mind to LLM agents, providing a formal definition of "Operational Morphology."
2. **Unified self-evolution framework**: uses the "Inputs / Agent System / Environment / Optimisers" four-element feedback loop to unify all self-evolution techniques.
3. **Cross-component coordination**: for the first time systematically studies the coordinated self-evolution of prompts / tools / memory / code.
4. **Reference implementation**: the open-source MorphAgent reference system covering implementation, evaluation protocols, and observability solutions.
5. **Governance solutions**: formal schemes of Verifiable Self-Modification and Provable Improvement for self-modifying systems.
