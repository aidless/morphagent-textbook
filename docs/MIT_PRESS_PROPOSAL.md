## 2. 目标读者

本书面向以下三类读者：

1. **研究生与高年级本科生**（计算机科学、人工智能、认知科学方向）：系统学习 LLM Agent 的理论基础与工程实践，课程教材或自学参考。
2. **AI 工程师与研究员**（工业界与学术界）：深入理解自修改 Agent 的形态可塑性、协同进化、安全性验证等前沿课题，构建下一代自适应智能系统。
3. **认知科学家与哲学家**：了解 4E 认知理论（具身、嵌入、延伸、生成）如何从生物认知映射到人工 Agent，形成跨学科对话。

**先修要求**：具备 Python 编程基础，了解机器学习基本概念。无需深入认知科学背景（第 II 部分从零引入 4E 认知理论）。

## 3. 书籍简介

当大语言模型（LLM）获得调用外部工具和执行代码的能力时，"语言模型"就跨越了"智能体"的边界。本书提出的"操作形态学"（Operational Morphology）正是对这一跨越的理论刻画：Agent 的 Prompt 可以自我修改，工具可以自动创建，记忆结构可以自适应调整，代码可以自主进化。这些维度构成了一种"形态可塑性"（morphological plasticity）——Agent 的认知架构不再由人类工程师预先固定，而是在与环境的交互中持续重塑。

本书将这一技术现象与 4E 认知理论（具身认知 Embodied Cognition、嵌入认知 Enacted Cognition、延伸认知 Extended Mind、生成认知 Enactivism）进行系统性对接。全书分为六大部分：第一部分建立 LLM Agent 的工程基础；第二部分引入 4E 认知的理论框架；第三部分提出操作形态学的核心理论，包括形态空间的形式化定义、协同进化动力学、元控制器架构；第四部分深入高级主题——可验证自修改、形态鲁棒性、调试可观测性；第五部分讨论部署运维与安全对抗；第六部分包含术语表、数学附录、工具索引和配套实验代码。

全书 25 章正文 + 6 个附录，约 800 页，200,000 字，31 个配套实验。每一章均包含学习目标、正文、本章小结、推荐阅读、5-10 道练习题和参考文献。代码实验覆盖基线评估、工具描述工程、Prompt 进化、记忆适应、代码进化、协同进化、元控制器和安全性测试等完整链条。

## 4. 竞争作品分析

| 对比维度 | 本书 | 《Human Compatible》(Russell) | 《Artificial Intelligence: A Modern Approach》(Russell & Norvig) | 《Surfing Uncertainty》(Clark) |
|---|---|---|---|---|
| 核心主题 | 自修改 Agent 的认知架构 | AI 安全与价值对齐 | 通用 AI 教科书 | 预测加工与具身认知 |
| 是否覆盖 4E 认知 | 系统对接（第 II 部分全部） | 不涉及 | 不涉及 | 涉及但未连接 AI 系统 |
| 是否覆盖 LLM Agent | 全书核心 | 不涉及 | 仅传统 Agent | 不涉及 |
| 自修改/自进化 | 核心理论贡献 | 不涉及 | 不涉及 | 不涉及 |
| 配套代码/实验 | 31 个可运行实验 | 无 | 有限 | 无 |
| 面向读者 | 工程师 + 认知科学家 | 通用读者 | 学生 | 认知科学研究者 |
| 差异化优势 | **唯一一本将 4E 认知与 LLM Agent 自修改架构系统性对接的教科书** | 经典安全论述 | 权威但无自修改内容 | 理论深度但缺工程 |

**本书的独特定位**：填补了"4E 认知理论"与"LLM Agent 工程"之间的空白。现有教材要么只讲传统 AI（Russell & Norvig），要么只讲认知科学理论（Clark, Varela），要么只讲 LLM 工程实践（各类技术书籍）。本书是第一本以"操作形态学"为理论框架，将认知科学理论与自修改 Agent 工程实践系统性统一的教科书。
